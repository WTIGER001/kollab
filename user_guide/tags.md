# User Guide: Document Tagging (Labels)

This guide describes how to use and manage tags (labels) in Project Arkollab to categorize and search documents.

---

## 🏷️ 1. Adding and Removing Tags on a Page

Tags can be attached to any page from the page footer, right above the comments section. This action is available in **both Read and Edit modes**.

### Adding a Tag:
1. Scroll to the bottom of the page. Under the **Page Tags** header, you will see existing tag chips and a small `+` (Add Tag) icon button.
2. Click the `+` button. This opens a popover search bar.
3. Type the name of the tag you wish to add:
   - If the tag **already exists**, click the tag chip in the list to attach it to the page.
   - If the tag **does not exist**, click the **Create "[tag-name]"** button to create it globally and attach it immediately.
4. The tag will be attached, styled according to its color category.

### Removing a Tag:
1. Locate the tag chip you want to remove inside the **Page Tags** footer.
2. Click the small `x` (Delete) button on the right side of the chip.
3. The tag will be instantly detached from the page.

---

## ⚙️ 2. Global Tag Management Page

You can manage all tags globally in your workspace by navigating to the **Manage Tags** page:
1. Click **Manage Tags** (Tag icon) at the bottom of the sidebar, open your user profile dropdown menu in the top navbar and click **Manage Tags**, or click **Go to Tag Manager** under the **Manage Tags** card within Team Settings, Project Settings, or Personal Settings.
2. This opens the tag management dashboard.

From this page, you can perform the following actions:
- **Add Tag**: Click the **Add Tag** button at the top right, choose a name, write a definition/description, select a color theme (Purple, Blue, Green, Red, Orange, Gray), and click **Save**.
- **Edit Tag**: Click the **Pencil icon** on any tag card to modify its name, edit its definition, or change its color category.
- **Delete Tag**: Click the **Trash icon** on any tag card to globally delete the tag from the system. *Warning: Deleting a tag removes it from all pages. This action cannot be undone.*
- **Usage Statistics**: Each tag card displays the number of pages currently labeled with that tag.
